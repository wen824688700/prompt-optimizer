## Toast样式规格

### 尺寸和位置
- **位置**: 顶部居中
- **距离顶部**: 16px (1rem)
- **内边距**: 水平 24px, 垂直 12px
- **圆角**: 8px (rounded-lg)

### 颜色
- **背景色**: #16a34a (green-600)
- **文字颜色**: #ffffff (white)
- **阴影**: shadow-lg

### 图标
- **尺寸**: 20x20px (w-5 h-5)
- **类型**: 勾选图标 (checkmark)
- **颜色**: 白色

### 文字
- **字体大小**: 14px (text-sm)
- **字重**: 500 (font-medium)
- **内容**: 
  - 复制: "复制成功"
  - 保存: "保存成功"

### 动画
- **类型**: 淡入 + 向下滑动
- **时长**: 0.3秒
- **缓动**: ease-out
- **显示时长**: 2秒后自动消失

## 用户交互流程

### 复制流程
```
用户点击"复制"按钮
    ↓
复制内容到剪贴板
    ↓
显示"复制成功"Toast (顶部居中)
    ↓
0.3秒淡入动画
    ↓

    ↓

```

```
    ↓
0.3秒淡入动画
    ↓
保持显示2秒
    ↓
自动消失
```

## CSS实现细节

### 居中定位
```css
.toast {
  position: fixed;
  top: 1rem;              /* 距离顶部16px */
  left: 50%;              /* 左边缘在50%位置 */
  transform: translateX(-50%);  /* 向左移动自身宽度的50% */
  z-index: 50;            /* 确保在最上层 */
}
```

### 动画效果
```css
@keyframes fade-in {
  from {
    opacity: 0;
    transform: translateY(-10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.animate-fade-in {
  animation: fade-in 0.3s ease-out;
}
```

### 完整样式
```css
.toast {
  position: fixed;
  top: 1rem;
  left: 50%;
  transform: translateX(-50%);
  background-color: #16a34a;  /* green-600 */
  color: white;
  padding: 0.75rem 1.5rem;
  border-radius: 0.5rem;
  box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1);
  display: flex;
  align-items: center;
  gap: 0.5rem;
  animation: fade-in 0.3s ease-out;
  z-index: 50;
}
```

## 可访问性考虑

### ARIA属性
```tsx
<div 
  role="alert"
  aria-live="polite"
  className="fixed top-4 left-1/2 transform -translate-x-1/2 ..."
>
  <span className="text-sm font-medium">复制成功</span>
</div>
```

### 键盘导航
- Toast不需要键盘交互（自动消失）
- 不会阻止用户继续操作

### 屏幕阅读器
- 使用 `role="alert"` 让屏幕阅读器自动读取
- 使用 `aria-live="polite"` 避免打断用户
